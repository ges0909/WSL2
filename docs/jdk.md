# JDK and Maven

## JDK

```bash
sudo apt install -y openjdk-21-jdk
```

## Maven

```bash
sudo apt install maven # optional or use wrapper './mvnw'
mvn -N io.takari:maven:wrapper # create a maven wrapper
```
